# RCH AI logo & branding assets

Official logo files for the rch ai GitHub organisation and related products.

## Files

| File | Use |
|------|-----|
| `rch_ai_logo_horizontal_black.*` | Horizontal logo on light backgrounds |
| `rch_ai_logo_horizontal_white.*` | Horizontal logo on dark backgrounds |
| `rch_ai_logo_stacked_black.*` | Stacked logo on light backgrounds |
| `rch_ai_logo_stacked_white.*` | Stacked logo on dark backgrounds |
| `favicon.*` | Browser tab / app icon |
| `rch_ai_logo_branding_package.zip` | Full package download |

Prefer SVG for web and print where supported; use PNG when raster is required.

## Usage

Use these assets for rch ai repos, documentation, and approved internal or external materials. Do not alter colours, proportions, or spacing without design approval.
